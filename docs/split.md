## Options

Find all the options using `teffects split -h`.

## Examples

```sh
$ teffects split -t AWESOME -w 800 -e 200 -d ~/Downloads
$ teffects split -t "AWESOME" -c "#4465aa" -b "#f4e990" -w 900 -e 500 -d ~/Downloads
```
