This is a CSS clip animation effect and it will open a browser.

## Options

Find all the options using `teffects clip-animate -h`.

## Examples

```sh
$ teffects clip-animate -t "TEFFECTS CLIP ANIMATE" -v "NEW CSS ANIMATION" -u "FOR YOUR ARTICLE"
```
