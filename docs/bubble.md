# Bubble

This is a bubble animation effect and it will open a browser.

## Options

Find all the options using `teffects bubble -h`.

## Examples

```sh
$ teffects bubble -t "long lorem ipsum and more" +i -c "#ff2d2d" -b "#008ac1"
```
